# LaTeX Display Support in LiquidVictor

LiquidVictor supports rendering LaTeX mathematical expressions in RevealJS presentations. LaTeX is the industry-standard typesetting language for mathematics, physics, and other sciences. When used in slide content, LaTeX notation is converted to beautifully rendered mathematical expressions in the final HTML presentation.

---

## How It Works

LiquidVictor uses two complementary libraries for LaTeX rendering:

1. **[Markdig](https://github.com/xoofx/markdig)** — A fast, powerful Markdown processor for .NET. Its mathematics extension (included via `UseAdvancedExtensions()`) converts LaTeX delimiters in Markdown content into HTML `<span>` and `<div>` elements with MathJax-compatible notation.

2. **[MathJax](https://www.mathjax.org/)** — A JavaScript engine used by the bundled RevealJS math plugin (`plugin/math/math.js`). The bundled file is the RevealMath plugin, which loads MathJax from a CDN by default; MathJax then reads the notation inserted by Markdig and renders it as high-quality, scalable mathematical typography in the browser. This means initial math rendering requires network access unless MathJax is configured to be served locally.

### Processing Pipeline

```
Markdown content with $LaTeX$
        ↓  (Markdig with UseAdvancedExtensions)
<span class="math">\(LaTeX\)</span>
        ↓  (MathJax via RevealMath plugin)
Rendered mathematical expression in the browser
```

No additional configuration is required — LaTeX support is enabled automatically in all RevealJS presentations generated by LiquidVictor.

## Syntax

### Inline Math

Wrap LaTeX expressions in single dollar signs (`$...$`) to render them inline within a paragraph.

**Markdown input:**
```markdown
Einstein's famous equation is $E = mc^2$, where $c$ is the speed of light.
```

**Rendered output:**  
Einstein's famous equation is *E = mc²*, where *c* is the speed of light.

---

### Display (Block) Math

Place double dollar signs (`$$`) on their own lines above and below an expression to render it as a centered display block.

**Markdown input:**
```markdown
The quadratic formula is:

$$
\frac{-b \pm \sqrt{b^2 - 4ac}}{2a}
$$
```

**Rendered output:**  
A large, centered display of the quadratic formula.

> **Note:** The opening `$$` and closing `$$` must each be on their own line for block display to work correctly.

## Usage Examples

### Example 1 — Basic Algebra

```markdown
The Pythagorean theorem states that $a^2 + b^2 = c^2$ for a right triangle.

For the general case, solve for $c$:

$$
c = \sqrt{a^2 + b^2}
$$
```

---

### Example 2 — Calculus (Derivatives and Integrals)

```markdown
The derivative of $f(x) = x^n$ is $f'(x) = nx^{n-1}$.

The definite integral of $f(x) = x^2$ from 0 to 1 is:

$$
\int_0^1 x^2 \, dx = \frac{x^3}{3} \Bigg|_0^1 = \frac{1}{3}
$$
```

---

### Example 3 — Greek Letters and Symbols

```markdown
Common Greek letters in mathematics include:

- $\alpha$, $\beta$, $\gamma$ (alpha, beta, gamma)
- $\theta$, $\phi$, $\psi$ (theta, phi, psi)
- $\pi \approx 3.14159$ (pi)
- $\infty$ (infinity)
- $\sum$, $\prod$, $\int$ (sum, product, integral)
```

---

### Example 4 — Summation and Series

```markdown
The sum of the first $n$ natural numbers is:

$$
\sum_{i=1}^{n} i = \frac{n(n+1)}{2}
$$

The geometric series with ratio $r$ converges when $|r| < 1$:

$$
\sum_{k=0}^{\infty} r^k = \frac{1}{1-r}
$$
```

---

### Example 5 — Matrices and Linear Algebra

```markdown
A 2×2 matrix is written as:

$$
\mathbf{A} = \begin{pmatrix} a & b \\ c & d \end{pmatrix}
$$

Its determinant is $\det(\mathbf{A}) = ad - bc$.

The identity matrix is:

$$
\mathbf{I} = \begin{pmatrix} 1 & 0 \\ 0 & 1 \end{pmatrix}
$$
```

---

### Example 6 — Physics and Special Relativity

```markdown
The Lorentz factor is defined as:

$$
\gamma = \frac{1}{\sqrt{1 - \frac{v^2}{c^2}}}
$$

Time dilation gives $t' = \gamma t$, and length contraction gives $L' = \frac{L}{\gamma}$.
```

---

### Example 7 — Statistics and Probability

```markdown
The normal distribution probability density function is:

$$
f(x) = \frac{1}{\sigma\sqrt{2\pi}} \, e^{-\frac{(x - \mu)^2}{2\sigma^2}}
$$

where $\mu$ is the mean and $\sigma$ is the standard deviation.
```

---

### Example 8 — Aligned Equations

Use `aligned` to show step-by-step derivations:

```markdown
$$
\begin{aligned}
  (a + b)^2 &= (a + b)(a + b) \\
            &= a^2 + ab + ba + b^2 \\
            &= a^2 + 2ab + b^2
\end{aligned}
$$
```

## LaTeX Quick Reference

| Notation | LaTeX Code | Example |
|----------|-----------|---------|
| Superscript | `^` | `$x^2$` → *x²* |
| Subscript | `_` | `$x_i$` → *xᵢ* |
| Fraction | `\frac{num}{den}` | `$\frac{1}{2}$` → *½* |
| Square root | `\sqrt{...}` | `$\sqrt{9}$` → *√9* |
| Summation | `\sum_{from}^{to}` | `$\sum_{i=0}^{n}$` |
| Integral | `\int_{a}^{b}` | `$\int_0^1$` |
| Infinity | `\infty` | `$\infty$` |
| Greek (lowercase) | `\alpha`, `\beta`, `\theta`, etc. | `$\theta$` |
| Greek (uppercase) | `\Sigma`, `\Pi`, `\Omega`, etc. | `$\Sigma$` |
| Bold | `\mathbf{...}` | `$\mathbf{x}$` |
| Hat (vector) | `\hat{...}` | `$\hat{x}$` |
| Dot (derivative) | `\dot{...}` | `$\dot{x}$` |
| Absolute value | `\lvert ... \rvert` | `$\lvert x \rvert$` |
| Norm | `\lVert ... \rVert` | `$\lVert x \rVert$` |
| Matrix | `\begin{pmatrix}...\end{pmatrix}` | see Example 5 |
| Cases | `\begin{cases}...\end{cases}` | see below |

**Cases example:**
```markdown
$$
f(x) = \begin{cases}
  x^2 & \text{if } x \geq 0 \\
  -x  & \text{if } x < 0
\end{cases}
$$
```

## Implementation Details

### Markdig Pipeline

The Markdig pipeline in `LiquidVictor.Output.RevealJs.Generator.Engine` is built with `UseAdvancedExtensions()`, which includes the mathematics extension:

```csharp
var pipeline = new MarkdownPipelineBuilder()
    .UseAdvancedExtensions()   // includes UseMathematics()
    .Build();
```

The mathematics extension handles:
- **Inline math** — `$...$` → `<span class="math">\(...\)</span>`
- **Display math** — `$$\n...\n$$` → `<div class="math">\[...\]</div>`

### RevealJS Math Plugin

The RevealJS template (`Templates/RevealJS/index.html`) loads `plugin/math/math.js`, which provides the `RevealMath` plugin. This plugin loads MathJax from a CDN and renders all `\(...\)` and `\[...\]` expressions found in the presentation HTML.

The plugin is initialized as part of `Reveal.initialize`:

```javascript
plugins: [ RevealHighlight, RevealMarkdown, RevealSearch, RevealNotes, RevealMath, RevealZoom ]
```

> **Network requirement:** MathJax is loaded from the CDN `https://cdn.jsdelivr.net/npm/mathjax@2/`. An internet connection is required when viewing presentations that contain LaTeX expressions.
